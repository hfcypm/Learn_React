# React 技能学习路线图（新手 → 进阶）

> 从 React 零基础到企业级进阶开发的完整技能栈，按学习阶段划分，帮你精准进阶、不踩坑。

## 学习阶段总览

| 阶段 | 名称 | 目标 |
|------|------|------|
| 一 | React 基础阶段 | 能独立开发简单页面、理解 React 核心思想 |
| 二 | React 进阶基础 | 解决复杂场景问题、优化代码、掌握高频 Hooks |
| 三 | React 生态与工程化 | 适配企业开发流程、掌握主流工具链 |
| 四 | React 高级进阶 | 解决大型项目难题、掌握性能调优、架构设计 |
| 五 | 附加技能 | 加分项技能扩展 |

## 阶段成果对照

- **阶段一成果**：能开发登录页、列表页、简单表单、TodoList 等单页面应用
- **阶段二成果**：能开发多页面应用、复杂表单、带权限的页面、优化渲染性能的项目
- **阶段三成果**：能开发企业级中后台系统、带数据缓存的应用、TS 强类型规范项目
- **阶段四成果**：能主导大型 React 项目架构、性能优化、团队代码规范、技术方案选型
